在文章末尾显示上次更新时间。

VitePress通过读取每个文件最近一次 Git 提交的时间戳来显示“上次更新”时间，所以这实际上不是真正的上次更新时间，而是`上次提交时间`。

```js
export default {
  lastUpdated: true
}
```

可以关闭指定页面的最后编辑时间。

```md
---
lastUpdated: false
---
```
