**教程视频：[Bilibili](https://www.bilibili.com/video/BV1ULCCBVEwe/?share_source=copy_web&vd_source=8c81e21b0ed380fd14142549f249565b)**

## 1.克隆Github仓库

打开仓库： [CloudFlare-ImgBed](https://github.com/MarSeventh/CloudFlare-ImgBed)，点击右上角的`复刻(Fork)`，将该仓库复刻到自己的Github账号下

![](https://imghub.mcoo.top/file/利用CloudFlare-ImgBed项目搭建图床/1765273843362_1.png)

## 2.打开Cloudflare

打开`Cloudflare`，注册账号（可使用Github账号关联）

点击右上角的个人简介，将主页修改为中文

![](https://imghub.mcoo.top/file/利用CloudFlare-ImgBed项目搭建图床/1765273854474_2.png)

## 3.部署Pages

在左侧侧边栏打开：`构建`→`计算和AI`→`Workers 和 Pages`

![](https://imghub.mcoo.top/file/利用CloudFlare-ImgBed项目搭建图床/1765273855188_3.png)

点击右上角：创建应用程序

选择：`Connect Github`

选择：仅选定的仓库，在下拉列表中选择刚刚复刻的仓库

点击：安装与授权

![](https://imghub.mcoo.top/file/利用CloudFlare-ImgBed项目搭建图床/1765273856464_4.png)

此时会跳转回创建`Worker`界面

点击最下方小字的`Get started`

![](https://imghub.mcoo.top/file/利用CloudFlare-ImgBed项目搭建图床/1765273862984_5.png)

点击导入现有Git储存库后面的开始使用

![](https://imghub.mcoo.top/file/利用CloudFlare-ImgBed项目搭建图床/1765273871614_6.png)

勾选刚刚授权的仓库，点击开始设置

![](https://imghub.mcoo.top/file/利用CloudFlare-ImgBed项目搭建图床/1765273877140_7.png)

项目名称可自定义

在构建命令处填写：`npm install`

其他保持不变，点击保存并部署

![](https://imghub.mcoo.top/file/利用CloudFlare-ImgBed项目搭建图床/1765273873851_8.png)

等待构建完成后，可以点击链接预览一下

::: warning
如果只显示Hellow World，那说明你有一步做错了，或者是将Pages部署成了Workers
:::

![](https://imghub.mcoo.top/file/利用CloudFlare-ImgBed项目搭建图床/1765273880549_9.png)

一切正常，点击继续处理项目即可

## 4.配置Workers KV

在左侧侧边栏打开：`构建`→`存储和数据库`→`Workers KV`

![](https://imghub.mcoo.top/file/利用CloudFlare-ImgBed项目搭建图床/1765273894742_10.png)

点击右上角`Create Instance`

![](https://imghub.mcoo.top/file/利用CloudFlare-ImgBed项目搭建图床/1765273893120_11.png)

命名空间名称`必须`填写：`img_url`，点击创建

![](https://imghub.mcoo.top/file/利用CloudFlare-ImgBed项目搭建图床/1765273900041_12.png)

## 5.添加资源绑定

回到`Workers 和 Pages`界面

点击自己刚刚部署的Pages

点击`设置`→`绑定`

![](https://imghub.mcoo.top/file/利用CloudFlare-ImgBed项目搭建图床/1765273900910_13.png)

点击添加，选择`KV命名空间`

![](https://imghub.mcoo.top/file/利用CloudFlare-ImgBed项目搭建图床/1765273906419_14.png)

变量名称`必须`填写`img_url`

KV命名空间选择`img_url`，点击保存

![](https://imghub.mcoo.top/file/利用CloudFlare-ImgBed项目搭建图床/1765273912449_15.png)

回到部署，选择重试部署

![](https://imghub.mcoo.top/file/利用CloudFlare-ImgBed项目搭建图床/1765273913285_16.png)

通过 Cloudflare 分配的域名（例如 xxx.pages.dev）加上 /dashboard 后缀，就可以访问管理后台，请设置一个密码
