### 【注意事项】

1. 可用鼠标点击任务查看详情，或点击加号快速添加新任务
2. 可用<sapn class="marker-evy"><任务名称>.<子任务名称></sapn>的形式来访问子任务
3. 命名子任务时，其名称<sapn class="marker-text">不可包含小数点</sapn>（即.）
4. **任务名称/旧任务名称**应与<sapn class="marker-evy">`!!task list`</sapn>中列出的一致

---

### 【命令列表】

`!!task overview` **显示任务概览**

`!!task help` **显示帮助信息**

- 显示此列表

`!!task list`**显示任务列表**

- 显示所有待完成任务

`!!task reload`**重载该插件**

`!!task detail <任务名称>`**查看任务详细信息**

`!!task list-all`**列出完整的任务表**

`!!task add <任务名称> [任务描述(可选)]`**添加任务**

- 例：`!!task add 一号空置域 史莱姆农场`
- 添加子任务：`!!task add 一号空置域.机器建造`

`!!task del <任务名称>` **删除任务**

`!!task rename <旧任务名称> <新任务名称>`**重命名任务**

`!!task change <任务名称> <新任务描述>`**修改任务描述**

`!!task done <任务名称>`**标注任务为已完成**

`!!task undone <任务名称>`**标注任务为未完成**

`!!task deadline <任务名称> <工期(日)>`**为任务设置工期**

- 例：`!!task deadline 【年镇装修】马年快乐！ 19`
- 工期按天数`ddl日期-设置任务日期`计算

`!!task player <玩家>`**查看玩家任务列表**

- 例：`!!task player sIqI_101111`

`!!task priority <任务名称> <优先级>`**为任务设置优先级**

- 例：`!!task priority 【年镇装修】马年快乐！ 1`

`!!task responsible <任务名称> <玩家>`**设置任务的责任人**

- 例：`!!task responsible 科隆大教堂装修 sIqI_101111`

`!!task unresponsible <任务名称> <玩家>`**移除任务的责任人**

- 例：`!!task unresponsible 科隆大教堂装修 sIqI_101111`
